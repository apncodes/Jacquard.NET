# Samples

All samples are runnable projects in the [`samples/`](https://github.com/apncodes/Jacquard.NET/tree/main/samples) directory. Each demonstrates a specific pattern or integration.

## Getting Started

| Sample | Description |
|---|---|
| [CliAgent](https://github.com/apncodes/Jacquard.NET/tree/main/samples/CliAgent) | Minimal multi-turn streaming REPL — the simplest possible agent |
| [QuickstartSample](https://github.com/apncodes/Jacquard.NET/tree/main/samples/QuickstartSample) | The code from the Getting Started page as a runnable project |
| [DiAgent](https://github.com/apncodes/Jacquard.NET/tree/main/samples/DiAgent) | Full dependency injection wiring with `IServiceCollection` |
